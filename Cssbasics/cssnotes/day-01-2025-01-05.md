## Button States (Pseudo-classes)
- Used `:hover` to style elements on mouse-over
- Used `:active` to style elements while clicking
- Improves user feedback and interactivity
```css
button:hover {
  background-color: blue;
}

button:active {
  background-color: darkblue;
}
```

---

## Padding

Padding controls the space between the content of an element and its border.

- Increasing padding makes buttons easier to click
- Padding contributes to the total size of an element

**Padding properties:**
- `padding-top`
- `padding-right`
- `padding-bottom`
- `padding-left`

**Difference from margin:**  
Padding adds space inside the element, while margin adds space outside.

---

## Transitions

Transitions allow CSS properties to change smoothly instead of instantly.

- Often used with `:hover`
- Makes UI interactions feel natural

**Syntax:**
```css
transition: property duration timing-function delay;
```

**Example:**
```css
button {
  transition: background-color 0.3s;
}
```

## Chrome DevTools

Chrome DevTools is a browser tool used for debugging and inspecting web pages.

**Used DevTools to:**
- Inspect HTML elements
- Modify CSS live in the browser
- Analyze margin, padding, and border
- View computed styles to understand final applied values

**Why this matters:**  
DevTools help debug layout issues and understand how CSS actually behaves.
