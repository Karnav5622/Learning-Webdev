## Text Styling
Styled text using various CSS properties:
- `color` - Sets text color
- `font-size` - Controls text size
- `font-family` - Specifies font type
- `font-weight` - Controls text boldness
- `text-align` - Aligns text (left, center, right)
- `margin-top` - Space above element
- `margin-bottom` - Space below element

---

## HTML Text Elements

### `<strong>`
- Makes text bold
- Has semantic meaning (indicates importance)
- Recommended for important content

**Example:**
```html
<p>This is <strong>very important</strong> information.</p>
```

---

### `<u>`
- Creates underlined text
- Mainly visual, not semantic
- Use sparingly (can be confused with links)

**Example:**
```html
<p>This word is <u>underlined</u>.</p>
```

---

### `<span>`
- Inline container element
- No default styling
- Used to style specific text portions or group inline elements

**Example:**
```html
<p>This is <span class="highlight">important</span> text.</p>
```

**CSS:**
```css
.highlight {
  background-color: yellow;
  font-weight: bold;
}
```

**Why use `<span>`?**  
It allows you to apply CSS styles to specific parts of text without breaking the flow or creating new lines.

---

## Key Takeaways
- Use semantic HTML tags (`<strong>`) when meaning matters
- Use `<span>` for styling without semantic meaning
- Text styling properties give you full control over appearance
- Combine HTML elements with CSS classes for powerful styling
