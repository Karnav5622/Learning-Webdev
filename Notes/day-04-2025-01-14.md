### ğŸ—ï¸ HTML Text Elements

#### **Headings** (For Titles and Sections)
Headings create structure on your page, like chapters in a book:

```html
<h1>Main Title</h1>          <!-- Biggest -->
<h2>Section Title</h2>       <!-- Second biggest -->
<h3>Subsection Title</h3>    <!-- Third biggest -->
```
**ğŸ’¡ Think of it like:** H1 = Book title, H2 = Chapter, H3 = Section

---

#### **Paragraphs** (For Regular Text)
```html
<p>This is a paragraph of text.</p>
```
**ğŸ’¡ Use for:** Any block of regular text

---

#### **Bold Text** (Two Ways)

```html
<b>This is bold</b>              <!-- Just looks bold -->
<strong>This is important</strong>  <!-- Bold + tells screen readers it's important -->
```
**ğŸ’¡ When to use which:**
- Use `<strong>` for important information (like warnings)
- Use `<b>` just to make text look bold (like product names)

---

#### **Italic Text** (Two Ways)

```html
<i>This is italic</i>           <!-- Just looks italic -->
<em>This is emphasized</em>     <!-- Italic + tells screen readers to emphasize -->
```
**ğŸ’¡ When to use which:**
- Use `<em>` when you want to stress something (like "really" important)
- Use `<i>` for things like book titles or foreign words

---

#### **Block vs Inline Elements**

**Block elements** = Take up the full width (like stacking boxes)
- Examples: `<div>`, `<p>`, `<h1>`
- Each starts on a new line

**Inline elements** = Sit next to each other (like words in a sentence)
- Examples: `<span>`, `<a>`, `<strong>`
- Don't break to a new line

```html
<p>This is a paragraph <strong>with bold text</strong> inside it.</p>
```

### ğŸ¨ CSS Text Styling

CSS controls how your text looks visually.

#### **Font, Size, and Color**
```css
p {
  font-family: Arial, sans-serif;  /* Font type */
  font-size: 16px;                 /* Text size */
  color: #333333;                  /* Text color (dark gray) */
}
```

#### **Text Alignment**
```css
p {
  text-align: center;   /* Options: left, right, center, justify */
}
```

#### **Spacing**
```css
p {
  line-height: 1.5;        /* Space between lines */
  letter-spacing: 0.5px;   /* Space between letters */
}
```

**ğŸ’¡ Quick tip:** `line-height: 1.5` means 1.5 times the font size

### ğŸ  The Complete HTML Skeleton


---

### ğŸ” Breaking It Down

#### **1. `<!DOCTYPE html>`**
- Tells the browser: "This is an HTML5 document"
- Always put this at the very top

#### **2. `<html lang="en">`**
- The container for your entire page
- `lang="en"` means the page is in English

#### **3. `<head>` Section (Invisible Setup)**
Contains information ABOUT your page (not shown to visitors):

```html
<head>
  <meta charset="UTF-8">              <!-- Character encoding (supports all languages) -->
  <meta name="viewport" ...>           <!-- Makes page mobile-friendly -->
  <title>Page Title</title>           <!-- Shows in browser tab -->
  <link rel="stylesheet" href="...">  <!-- Links to your CSS file -->
</head>
```

**ğŸ’¡ Think of `<head>` as:** The settings and instructions for your page

#### **4. `<body>` Section (Visible Content)**
Everything visitors actually see goes here:

```html
<body>
  <h1>Welcome!</h1>
  <p>This text is visible on the page.</p>
</body>
```

## ğŸ¨ CSS Integration

### **Method 1: Internal Styles** (CSS inside HTML file)
```html
<head>
  <style>
    body {
      margin: 0;
      font-family: Arial, sans-serif;
    }
  </style>
</head>
```

### **Method 2: External Stylesheet** (Separate CSS file - RECOMMENDED)
```html
<head>
  <link rel="stylesheet" href="styles.css">
</head>
```

Then create `styles.css`:
```css
h1 {
  color: blue;
}

p {
  color: #444;
}
```

**ğŸ’¡ Why external is better:** Keep HTML and CSS separate, easier to maintain
